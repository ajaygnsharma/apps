Reboot iServer

This will remotely reset the power on the iServer. If the Reboot button is clicked a window will pop up with the message: WARNING: Are you SURE you want to reboot this device?

Restore Factory Defaults

This will force the iServer back to its factory default settings. If the Restore Factory Defaults button is clicked a window will pop up with the message: WARNING: Are you SURE you want to reset the configuration to the factory defaults? This device will be rebooted for the changes to take affect. Note: Changing the iServer to its factory settings can also be done using a dip switch on the iServer.

Upload New Firmware

This allows users to upgrade the iServer’s firmware over the network. By clicking on the Browse button, the firmware binary file can be found and selected. Clicking the Upload button will send the file to the iServer.
