Diagnostics

This option will allow the users to monitor the iServer’s operation and traffic through its Ethernet and serial ports.

Serial Port

The number of bytes received and transmitted in real time. The maximum is 9999. The status of signals (CTS, RTS, DSR, DCD, and DTR) shown in real time to be high or low.

Ethernet Port

The number of TCP, UDP, and ICMP bytes received and transmitted by the Ethernet port. The maximum is 9999.

Ping

To ping a network host either a DNS name or an IP Address should be specified.

Count

This is the number of ping packets to send by the iServer. The iServer will wait 5 seconds for a response for each ping packet sent before it times out.
